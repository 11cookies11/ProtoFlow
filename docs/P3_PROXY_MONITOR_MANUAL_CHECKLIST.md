# P3-01 代理监听手工回归清单

## 目标
- 覆盖代理监控核心流程：创建、编辑、删除、启停、抓包联动。
- 验证 UI 展示、后端状态和抓包状态一致。

## 前置条件
- 运行主程序：`python main.py`
- 打开页面：`代理监控`
- 已连接可用串口或可用模拟通道（推荐 COM mock 环境）

## 用例

| ID | 场景 | 步骤 | 期望 |
|---|---|---|---|
| PM-001 | 创建代理 | 新建代理对并保存 | 列表新增卡片，字段展示正确，出现“创建成功”提示 |
| PM-002 | 编辑代理 | 编辑代理名称与端口并保存 | 卡片内容更新，出现“保存成功”提示 |
| PM-003 | 启停代理 | 对同一卡片执行启用/停用 | 状态在 `configured/running/stopped` 间正确切换，出现对应提示 |
| PM-004 | 删除代理 | 删除代理并确认 | 卡片移除，出现“删除成功”提示 |
| PM-005 | 抓包开始 | 点击抓包并发送测试流量 | 抓包窗口有帧数据，卡片显示“抓包中” |
| PM-006 | 抓包停止联动 | 停用正在抓包的代理 | 抓包自动停止，状态更新为已停止 |
| PM-007 | 窗口切换 | 切换 10s/30s/60s 统计窗口 | 带宽数值和趋势线随窗口变化 |
| PM-008 | 通道一致性提示 | 抓包通道与代理端口不一致 | 抓包窗口显示不一致提示 |
| PM-009 | 统计区一致性 | 观察总数/运行中/已配置/异常统计 | 统计数字与当前卡片状态一致 |
| PM-010 | 刷新行为 | 点击“刷新状态” | 列表同步后端最新状态，出现刷新成功提示 |
